# DMRC

DMRC Metro Management & Analysis System

Designed and implemented a relational database system for managing metro stations, routes, passengers, smart cards, and ticket transactions.
Developed complex SQL queries, joins, views, stored procedures, and triggers to analyze metro operations and passenger travel patterns.
Performed data analysis tasks such as identifying most frequently used stations, revenue tracking, route utilization, and passenger movement trends.
Ensured database normalization, data integrity, and efficient query execution for large-scale transportation data.
Utilized MySQL for database design, data management, and analytical reporting.
